Collection
============
+ 동일한 타입의 객체를 담을 수 있는 객체
+ Set, Queue, Map, List등의 자료구조를 제공하여 사용


List
-------------
+ 순서가 있는 자료구조로 각 요소들을 index를 붙여서 관리한다.


ArrayList
-------------
+ 사이즈 변경이 가능한 Array
+  Array Length를 자유롭게 증감시킬수 있다.
