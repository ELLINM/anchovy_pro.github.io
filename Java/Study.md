Java Data Type
==================

자료형
--------------------
>기본 자료형 Primitive data type
= 정수형(byte, short, int, long), 실수형(float, double), 문자형(char=character), 논리형(boolean)

>참조 자료형 Reference data type
= 배열, 클래스 객체, 인터페이스


기본값
------------------------------
>number라는 class를 만들고 아무것도 넣지 않는다면
default => int 0, double 0.0, boolean false
