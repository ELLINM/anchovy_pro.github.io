Pakage
======
+ module의 합
+ Packages는 도트(.)를 사용하여 파이썬 모듈을 계층적(directory 구조)으로 관리할 수 있게 해줌


Module
------
+ Code가 들어있는 File
+ Code가 모여서 기능을 구현


__init__.py
---------
+ __init__.py file은 해당 directory가 pakage의 일부임을 알려주는 역할
+ pakage에 포함된 directory에 __init__.py file이 없다면 pakage로 인식되지 않음
+ (*) 은 all을 뜻함
